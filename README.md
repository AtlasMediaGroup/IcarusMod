#IcarusMod
###What is the _IcarusMod_?
The IcarusMod is an all-op Bukkit plugin created for the _**Superior Freedom Network**_.
####Features
 - MySQL Configuration Files
 - Custom Command Handling
 - Custom Displayname Handling
